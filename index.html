Creative Computing 2021/22

The activity introduces different ways on how to create games, music and art with computer programs.

Part Ia: Basic concepts of coding with Scratch

At the beginning we learn the basics of coding by doing small animations and games with Scratch.

Part Ib: Web design

Here we either learn the basics of web design with HTML and CSS or deepen our knowledge. Throughout the year we publish our work on a website on Github Pages. That's what you see right now.

Part II: Sound and music

We learn about audio, we create sound effects and simple music sequences using the program MAX 8.

Part III: Animation

We learn to animate characters on screen and create a short video with Alice 3.

Part IV: Coding and art

We use our knowledge to create a piece of art or a game combining hardware and software. We learn and use the languages P5.JS and Processing for that purpose.

By the end of the year, we will have had a good exposure to various areas at the intersection of arts and computer science. This website will serve as a blog to document my progress.
